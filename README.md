# CLI Personal Assistant (v1.0)

A command-line tool to manage daily tasks and check the weather, built with Python.

## Features

- **Task Manager**: Add, Edit, Delete, and Mark tasks as Done.
- **Weather**: Check current weather for any city and save search history.
- **Storage**: Uses SQLite/Postgres via SQLAlchemy.
- **UI**: Beautiful terminal interface using `rich` and `questionary`.

## How to Run

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
